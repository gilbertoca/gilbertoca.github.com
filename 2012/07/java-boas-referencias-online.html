Java - Boas referências online

by Gilberto C. Andrade on 13 Julho 2012

Tagged as: Estou lendo e você, Geral, Internet, Java,

Tenho alguns livros sobre Java, mas com as mudanças que a linguagem sofreu nos três últimos anos tenho por mim que minhas referências estão desatualizadas. Meu grande interesse é entender melhor os detalhes que muitos programadores (vindos de outra linguagem ou não) ignoram - seja por desconhecimento, seja por desinteresse, pois o arcabouço de APIs Java é muito amplo.

Enquanto realizo minha tarefa diária - implementando ou corrigindo código - costumo manter ao meu lado algumas referências e hoje vou postar as que costumo consultar via internet. Vamos a elas:

  • <a href=“http://www.gui.net/cs4448.html” title:“Object-Oriented Programming and Design with UML and Java ” target="_blank">Object-Oriented Programming and Design with UML and Java
  • <a href=“http://www.exampledepot.com/” title:“The Java Developers Almanac and more” target="_blank">The Java Developers Almanac and more - Author: Example Depot
  • <a href=“http://math.hws.edu/javanotes/” title:“Introduction to Programming Using Java, Sixth Edition” target="_blank">Introduction to Programming Using Java, Sixth Edition - Author: David J. Eck (eck@hws.edu)
  • <a href=“http://en.wikibooks.org/wiki/Java_Programming” title:“Java Programming” target="_blank">Java Programming - Authors & Contributors
  • Java Persistence - Author: James Sutherland, Doug Clarke
  • Introduction to Programming in Java - Author: Robert Sedgewick and Kevin Wayne
  • Thinking in Java - Author: Bruce Eckel
  • Java – Objects First: An Introduction to Computer Programming using Java and BlueJ - Author: Rick Gee
